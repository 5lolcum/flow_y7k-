# Flow Store - Tienda de Ropa Online

Plataforma de e-commerce moderna para vender ropa online con todas las características necesarias para emprender.

## 🚀 Características

- ✅ Catálogo de productos con filtros
- ✅ Carrito de compras
- ✅ Sistema de usuarios y autenticación
- ✅ Pasarela de pagos (Stripe)
- ✅ Panel de administrador
- ✅ Sistema de órdenes
- ✅ Reseñas y calificaciones
- ✅ Búsqueda avanzada
- ✅ Responsive design

## 📁 Estructura del Proyecto

```
flow_y7k-/
├── frontend/          # Aplicación Next.js
├── backend/           # API Node.js/Express
├── database/          # Scripts de BD
└── docs/              # Documentación
```

## 🛠️ Tech Stack

- **Frontend:** Next.js 14, React, TailwindCSS, TypeScript
- **Backend:** Node.js, Express, PostgreSQL
- **Autenticación:** JWT
- **Pagos:** Stripe
- **Hosting:** Vercel, Render

## 📋 Pasos para empezar

1. Clonar el repositorio
2. Instalar dependencias en frontend y backend
3. Configurar variables de entorno
4. Ejecutar migraciones de BD
5. Iniciar servidores
